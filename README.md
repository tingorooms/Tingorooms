# Room Rental Management Platform

Production-ready full-stack room rental platform for room discovery, posting, realtime chat, notifications, and admin operations.

## Product Overview

- Frontend: React + TypeScript + Vite
- Backend: Node.js + Express
- Database: MySQL-compatible (Planetscale)
- Realtime chat: Supabase
- Images/CDN: Cloudflare R2 (with optional fallback)
- Email: Brevo SMTP
